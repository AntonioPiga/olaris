Subcommand: nuv setup 

Usage:
  setup configure [--all] [--redis] [--mongo] [--minio] [--cron] [--static]
  setup --devcluster
  setup --current
  setup status
  setup clean

Options:
  --devcluster  deploy a local development cluster in docker
  --current     deploy in the current kubernetes cluster
  --all         enable all services
  --redis       enable redis
  --mongo       enable mongo
  --minio       enable minio
  --cron        enable cron
  --static      enable static

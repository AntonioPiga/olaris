Subcommand: nuv setup 

Usage:
  setup configure [--all] [--redis] [--mongo] [--minio] [--cron] [--static]
  setup cluster [<context>]
  setup local
  setup server <server> [<user>]
  setup status
  setup --devcluster

Commands:
  cluster       deploy Nuvolaris in the Kubernetes cluster using the <context>, default the current
  local         creates a local Kubernetes in docker and deploy Nuvolaris
  server        create a Kubernetes in server <server> and deploy Nuvolaris
                the server must be accessible with ssh using the <user>, default root

Options:
  --all         enable all services
  --redis       enable redis
  --mongo       enable mongo
  --minio       enable minio
  --cron        enable cron
  --static      enable static
  --devcluster  (legacy) same as local
